System Requirements

The minimum and recommended requirements for installing LibreOffice on a Windows system are as follows:

  • Microsoft Windows XP, Vista, Windows 7, or Windows 8;
  • Pentium-compatible PC (Pentium III, Athlon or more-recent system recommended);
  • 256 MB RAM (512 MB RAM recommended);
  • Up to 1.5 GB available hard disk space;
  • 1024x768 resolution (higher resolution recommended), with at least 256 colors.

Installation Instructions for Windows

1 Download the LibreOffice installer.

  • Save the Installer;
  • Double-click on the Installer;
  • A Welcome dialog box opens and click on "Next"

2 Choose the kind of Installation you want.

3 Tell LibreOffice if you want to create a shortcut on your desktop.

4 LibreOffice is installing.

5 LibreOffice is ready!
